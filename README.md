# 🌸 Bloom Expense Tracker

A modern and responsive **Expense Tracker Web Application** built using **HTML, CSS, and JavaScript**. Bloom helps users manage their personal finances by tracking income and expenses, analyzing spending patterns, and setting budgets with a clean and aesthetic user interface.

link of the website-https://blloom.netlify.app/
---

## ✨ Features

### 📊 Dashboard
- View Total Balance
- Track Total Income
- Monitor Total Expenses
- Display Recent Transactions

### 💳 Transaction Management
- Add new transactions
- Edit existing transactions
- Delete transactions
- Categorize transactions (Food, Shopping, Travel, Bills, Entertainment, etc.)

### 📜 Transaction History
- View complete transaction history
- Displays:
  - Transaction Title
  - Category
  - Amount
  - Date
  - Transaction Type

### 📈 Analytics
- Doughnut Chart for expense categories
- Bar Chart comparing Income vs Expenses
- Real-time financial insights

### 💰 Budget Planner
- Set monthly budget
- Track remaining budget
- Budget progress visualization
- Automatically updates based on expenses

### 💾 Local Storage
- Stores all transaction data using LocalStorage
- Data persists even after browser refresh
- No database required

### 📱 Responsive Design
- Desktop
- Tablet
- Mobile Friendly
